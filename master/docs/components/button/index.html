Button group

ButtonGroups keep a consistent flow of spacing between Buttons and sibling elements, so liek other inline elements they should always be used, even when there is only one button.

<ButtonGroup>
  <Button> ... </Button>
  <Button> ... </Button>
  <Button> ... </Button>
</ButtonGroup>

Joined button group

<ButtonGroup joined={true}>
  <Button style="secondary"> ... </Button>
  <Button style="secondary"> ... </Button>
  <Button style="secondary">
    <ButtonIcon name="twitter" /> </Button>
</ButtonGroup>
